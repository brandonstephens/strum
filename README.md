# Strum

Helping hand for learning to strum on the guitar.

Based on https://www.youtube.com/watch?v=CjM5fyXoV8w

## Installation

1. `nvm use`
2. `npm install`
3. `npm start`
4. <http://localhost:8081>

## Build

1. `npm run build`
